Ne laissez pas les caractères spéciaux ou les points et les traits vous rendre confus.

Le code Morse est un système pour encoder les lettres et les nombres sous forme de points et de traits.
La durée du trait est trois fois la durée d'un point.
Chaque point ou trait est suivi d'une période d'absence de signal, appelée espace, égale à la durée du point.

Le code Morse a premièrement été utilisé pour communiquer par télégraphe.
Le code Morse le plus connu est SOS, qui est utilisé pour signaler une urgence.
En plus d'écrire des points et des tirets, vous pouvez également signaler du code Morse en allumant et éteignant une lumière, en sonnant une corne ou en tapant, etc.
Tous les moyens de transmettre le code Morse exigent que le destinataire le comprenne.
Si ce n'est pas le cas, notre outil de chiffrement/déchiffrement peut être utilisé pour trouver le mot de passe, le numéro secret, ou toute info que vous voulez relayer .
Comme les gens communiquent rarement via le télégraphe, ou l'apprennent dans l'armée, très peu de gens ont mémorisé les codes morse.
Au lieu de cela, il est devenu un moyen de coder quelque chose à des fins spéciales.
C'est une langue utile à savoir même si vous voulez simplement apprendre la cryptographie ou résoudre des cryptogrammes pour le plaisir.
