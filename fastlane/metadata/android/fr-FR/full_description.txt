Qu'est-ce que le Morse ?
Le Morse est un système permettant d'encoder des lettres et des chiffres à l'aide de tirets et de points. La durée du tiret est trois fois la durée d'un point. Chaque point ou tiret est suivi d'une période d'absence de signal, appelée espace, égale à la durée du point. Le Morse a d'abord été utilisé pour communiquer par télégraphe. Le code Morse le plus connu est SOS, qui est utilisé pour signaler une urgence. Outre la saisie de points et de tirets, vous pouvez également utiliser le code Morse en allumant et en éteignant une lumière, en faisant retentir un klaxon ou en tapotant, et bien plus encore. Vous pouvez utiliser le code Morse pour transmettre facilement des messages par divers moyens si le destinataire comprend le code Morse. Si ce n'est pas le cas, ils peuvent utiliser notre outil de chiffrement pour trouver le mot de passe, le numéro secret ou toute autre information que vous souhaitez transmettre.
Comme les gens communiquent rarement par télégraphe, il y a aujourd'hui très peu de personnes qui peuvent mémoriser les codes Morse. Au lieu de cela, il est devenu un moyen d'encoder quelque chose ou des caractères spéciaux à des fins spéciales. Bien sûr, ces caractères spéciaux sont encore largement utilisés dans de nombreuses situations, vous ne savez pas quand vous en aurez vraiment besoin. C'est un langage utile à connaître même si vous voulez juste apprendre la cryptographie ou les cryptogrammes pour le plaisir.
