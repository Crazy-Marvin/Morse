O que é código Morse?
Código Morse é um sistema para codificar letras e números para traços e pontos.
O período de duração de traço é três vezes de duração de um ponto.
Cada ponto ou traço é seguido por um período de ausência de sinal, chamado de espaço, igual à duração do ponto.
O código Morse foi usado pela primeira vez para a comunicação via telégrafo.
O código Morse mais reconhecido é o SOS, que é usado para sinalizar uma emergência.
Além de digitar pontos e traços, você também pode sinalizar código Morse ligando e desligando a luz, soando uma buzina ou batendo entre outros.
Você pode usar o código Morse para transmitir mensagens facilmente através de vários meios se o destinatário entender o código Morse.
Caso contrário, eles podem usar nossa ferramenta decifradora para descobrir a senha, número secreto ou qualquer informação que você deseja transmitir.

Como as pessoas raramente se comunicam via telégrafo, hoje em dia há muito poucas pessoas que conseguem memorizar códigos Morse.
